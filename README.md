## ❗️ Troubleshooting

| Problem | Solution |
| :--- | :--- |
| **Driver failed to load** | Disable Secure Boot in your BIOS settings. |
| **Spoof not persisting** | Ensure you ran the "Clean" step before "Spoof". |
| **Antivirus blocking** | Add the BioSecure directory to your exclusions list. |
| **Blue Screen (BSOD)** | Ensure all Windows Updates are paused and Hyper-V is disabled. |
